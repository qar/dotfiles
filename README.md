# My VIM Configurations

## Use [vundle](https://github.com/VundleVim/Vundle.vim) to manage vim plugins

1. Download vundle
	`git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

2. Install Plugins
	`:PluginInstall`
